# Ielts-Listening-Reading-Record 雅思听阅成长记录
一款为雅思考生设计的听力&amp;阅读训练数据记录与分析工具
让每一次练习都“留有痕迹”，让每一分努力都“清晰可见”。

---

## ✨ 核心功能亮点

### 1. 🔄 三位一体的无缝切换模式
针对不同的备考场景，系统量身定制了三大核心模块：
- **🌿 碎片化刷题模式**：专为日常单项突破设计（如剑雅、机经或自定义材料）。支持精细到每一 Part 的正确率记录，并自动基于平均表现估算你的“综合等效分”。
- **🎓 全真套题模考模式**：高度还原真实考试节奏。不仅记录阅读和听力的各部分得分，还特别加入了**做题时间段记录**（如“上午黄金同频”、“午后抗困”），帮助你将大脑生物钟调整至最佳考场状态。
- **🔍 弱点定向复盘模式**：创新的“药丸式”切换开关，将刷题与套题中的错题自动归拢。通过打勾式的沉浸式复盘体验，清空待办列表，彻底告别“只刷不看”的无效努力。

### 2. 📊 智能可视化的成长看板
数据不应该只是冷冰冰的数字，系统内置了强大的可视化图表，让你的进步一目了然：
- **学习热力追踪**：自动绘制努力折线图，统计今日与累计的阅读/听力完成篇数，清晰看见自己的坚持。
- **多维能力雷达与走势**：套题模式下，通过雷达图对比“最新模考”与“历史均值”，直观暴露出当前的薄弱环节；搭配可滑动的分数走势折线图，精准把握提分节奏。
- **⚠️ 智能波动预警系统**：系统会在后台默默计算你各个 Part 的近期正确率与历史均值的差值。当某个题型表现出现大幅下滑时，系统会自动标红预警，提醒你“需重点复盘”。

### 3. 🎨 极致的 UI 美学与交互体验
- **莫兰迪治愈配色**：告别压抑的刷题氛围。系统采用莫兰迪绿（刷题/阅读）与靛蓝（套题/听力）作为主题色，搭配砖红色（复盘），辅以毛玻璃效果（Backdrop-blur）和丝滑的滚动条，带来极其舒适的视觉体验。
- **微交互与防错机制**：丝滑的页面切换动画、输错数据的红色抖动提醒、重复录入的智能拦截，每一个细节都在为你节省时间。
- **响应式设计**：无论是在宽屏电脑上沉浸式复盘，还是在手机上快速录入，界面都能完美适配。

### 4. ⏳ 温暖陪伴与数据自由
- **动态倒计时进度条**：首页顶部实时显示距考试天数与备考进度，搭配不同阶段的专属鼓励语，给你满满的仪式感与动力。
- **绝对的数据掌控权**：所有数据均通过 `localStorage` 安全存储在你的本地浏览器中。支持一键导出/导入完整的 JSON 备份文件，甚至可以导出 CSV 表格供 Excel 深度分析，数据永远属于你自己。
